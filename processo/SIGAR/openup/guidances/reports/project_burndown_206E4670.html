Main Description

O Relatório de Progresso do Projeto é a principal ferramenta para verificar o status do projeto. Ele mostra a tendência de quanto trabalho ainda deve ser feito no projeto. Isto é feito através da estimativa do esforço necessário para a conclusão de cada um dos Itens de Trabalho a serem tratados no projeto e mostrando como o esforço estimado está evoluindo de uma iteração à outra. O Relatório de Progresso do Projeto deve ser atualizado ao final de cada iteração.

A taxa de progresso do projeto é normalmente comunicada na forma gráfica. O gráfico do Relatório de Progresso do Projeto consiste de duas perspectivas, o eixo horizontal mostrando as iterações e o eixo vertical indicando os itens restantes da Lista de Itens de Trabalho. Adicionalmente a taxa de progresso médio das iterações precedentes é calculada e uma tendência para o restante do projeto é prevista a partir destes dados históricos.

O gerenciamento do Relatório de Progresso do Projeto é uma técnica que permite a correlação entre as metas da iteração e os Itens de Trabalho. O Gerente de Projeto usará as informações do relatório para comunicar o progresso e a tendência do projeto à gestão sênior.
