Example: Plano de Iteração
Este é um exemplo de um plano de iteração para uma pequena equipe. Neste exemplo, a equipe escolheu não relacionar os itens de trabalho no plano de iteração. Ao invés disto, a equipe irá procurar na Lista de Itens de Trabalho os itens atribuídos à iteração. Esta é a solução preferida quando os itens de trabalho são controlados em uma ferramenta que forneça capacidades básicas de pesquisa.
