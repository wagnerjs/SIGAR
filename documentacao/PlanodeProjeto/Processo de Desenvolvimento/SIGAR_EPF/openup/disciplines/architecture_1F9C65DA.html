Main Description

O Propósito desta disciplina é evoluir uma arquitetura robusta para o sistema.

A disciplina de Arquitetura está relacionada com outras disciplinas, da seguinte forma:

  • A disciplina de Requisitos fornece os requisitos arquiteturalmente significantes.
  • A disciplina de Desenvolvimento projeta e implementa a arquitetura.
  • A disciplina de Teste verifica a estabilidade e o quanto a arquitetura está correta.
  • A disciplina de Gestão de Projeto planeja o projeto e cada iteração.
Key Considerations

Criar e aplicar mecanismos arquiteturais é essencial para criar uma arquitetura robusta. Veja mais sobre mecanismos arquiteturais em Mecanismos Arquiteturais.
